## devtui jsonfmt

Format JSON

### Synopsis

Format JSON

```
devtui jsonfmt [flags]
```

### Examples

```

	devtui jsonfmt < testdata/example.json # Format and output to stdout
 	devtui jsonfmt < testdata/example.json > formatted.json # Output to file
	
```

### Options

```
  -h, --help   help for jsonfmt
```
