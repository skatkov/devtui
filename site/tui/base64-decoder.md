# Base64 Decoder

## Usage

1. Run `devtui` to open the main menu
2. Select "Base64 Decoder" from the list
3. Use the key bindings below to interact with the tool
4. Press `q` or `Ctrl+C` to return to the main menu

## Key Bindings

| Key | Action |
|-----|--------|
| `c` | copy text |
| `e` | edit base64 |
| `v` | paste base64 to decode |
| `q/ctrl+c` | quit |
